How this pass works

Automatic extraction: Name, dates, amount, type, source, and any placement clues the parser can find.
Placement field: Every record now includes a blank Program / Day / Day-part field you can fill by hand when the contract is vague.
Credit runs field: Every record also includes a blank Exact credit run dates / times field for the quarterly filing data you said you will refine later.
Conflict rule: Overlap flags only fire when the same underwriter overlaps itself. Different underwriters can overlap all day long.
Popup editing: Open any record from the Contracts view to edit it in a modal window.
